# 🎙️ Sermon Upload Server

Простой веб-сервер для загрузки проповедей в Telegram-канал церкви.

## Установка

### 1. Установите зависимости

```bash
cd server
npm install
```

или с использованием pnpm:

```bash
cd server
pnpm install
```

### 2. Настройте переменные окружения

Создайте файл `.env` в папке `server/`:

```env
TELEGRAM_BOT_TOKEN=8957414613:AAEFOs-4xAA6YUetbNHA16id5_14Ku2-JJU
TELEGRAM_CHANNEL_ID=-3565559545
PORT=3001
```

Или просто используйте значения по умолчанию (они уже установлены в коде).

## Запуск

### Режим разработки

```bash
npm start
```

### С автоматической перезагрузкой (требует nodemon)

```bash
npm run dev
```

## Использование

1. Откройте браузер на компьютере в церкви
2. Перейдите по адресу: `http://localhost:3001`
3. Заполните форму:
   - **Название проповеди** — обязательно
   - **Описание** — опционально
   - **Файл** — выберите аудио или видео
4. Нажмите "Загрузить"
5. Файл загружается в Telegram-канал
6. Проповедь появляется в приложении за несколько секунд

## Поддерживаемые форматы

**Аудио:**
- MP3
- WAV
- OGG
- MP4 (с аудиодорожкой)

**Видео:**
- MP4
- MOV
- AVI
- MKV

## Ограничения

- **Максимальный размер файла:** 2 ГБ
- **Максимальная длительность:** зависит от Telegram API

## Безопасность

⚠️ **Важно:** Токен бота хранится на сервере, а не на клиенте. Это безопасно, потому что:
- Операторы не видят токен
- Токен используется только на сервере
- Операторы могут только загружать файлы, но не могут удалять или изменять канал

## Развёртывание

### На локальной сети (церковь)

Сервер по умолчанию запускается на `localhost:3001`. Если вы хотите получить доступ с других компьютеров в сети:

1. Узнайте IP-адрес компьютера с сервером (например, `192.168.1.100`)
2. Откройте браузер на другом компьютере
3. Перейдите по адресу: `http://192.168.1.100:3001`

### На облачном сервере (опционально)

Для развёртывания на облачном сервере (Heroku, DigitalOcean и т.д.):

1. Установите Node.js на сервер
2. Скопируйте папку `server/`
3. Установите зависимости: `npm install`
4. Установите переменные окружения
5. Запустите: `npm start`

## Решение проблем

### "Ошибка при загрузке"

1. Проверьте интернет-соединение
2. Убедитесь, что бот добавлен в канал как администратор
3. Проверьте размер файла (не более 2 ГБ)

### "Файл не появляется в приложении"

1. Подождите несколько секунд
2. Обновите приложение (потяните вниз)
3. Проверьте, что файл загружен в Telegram-канал

### "Сервер не запускается"

1. Проверьте, что порт 3001 не занят
2. Убедитесь, что Node.js установлен: `node --version`
3. Проверьте логи ошибок в консоли
